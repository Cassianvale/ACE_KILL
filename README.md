### 如何打包

1. 安装依赖：  
```bash
pip install -r requirements.txt
```

2. 运行打包脚本：  
```bash
python build_exe.py
```

3. 打包完成后，将在项目根目录生成 `VALORANT_ACE_KILL.exe` 文件
